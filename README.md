# my-godot

Godot 4 project set up for Discord → VPS Hermes → GitHub → local Godot testing.

## Workflow

1. Ask Hermes in Discord to implement a small Godot change.
2. Hermes edits this repository on the VPS.
3. Hermes commits and pushes to GitHub.
4. On your local computer:

```bash
git pull
```

5. Open `project.godot` in Godot and test.
6. Send Godot console errors or feedback back to Discord.

## Local Test

Open this folder in Godot 4.x and run the main scene.

## VPS Project Path

```text
/opt/data/projects/my-godot
```
